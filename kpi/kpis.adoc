== Key performance indicators

=== Overview

The table below provides an overview of all KPIs that may be used in the quality assessment of a WCMP record.

.Table. WCMP Key Performance Indicators
|===
|Name|Properties|Description

|Good quality title
a|* ``properties.title``
|See <<_good_quality_title>>

|Good quality description
a|* ``properties.description``
|See <<_good_quality_description>>

|Time intervals
a|* ``time.interval``
* ``additionalElements.temporal.interval``
|See <<_time_intervals>>

|Graphic overview for metadata records
a|* ``$.links[?(@.rel=="preview")]``
|See <<_graphic_overview_for_metadata_records>>

|Links health
a|* ``links[*].href``
* ``properties.themes[].concepts[].url``
* ``properties.themes[*].scheme``
* ``properties.contacts[].links[].href``
|See <<_links_health>>

|Contacts
a|* ``properties.contacts[?(@.roles=="host")]``
``properties.contacts[?(@.roles=="host")].emails``
``properties.contacts[?(@.roles=="host")].contactInstructions``
``properties.contacts[?(@.roles=="publisher")]``
|See <<_contacts>>

|Persistent identifiers
a|* ``properties.externalIds``
|See <<_persistent_identifiers>>

|===
